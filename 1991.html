DESCRIZIONE DEL SISTEMA

Il sistema GESTIONE INCASSI rappresenta una soluzione informatica all'avanguardia per l'automazione dei processi di registrazione e controllo degli incassi aziendali.

Sviluppato utilizzando le piu' moderne tecnologie informatiche disponibili, il software e' progettato per funzionare su personal computer IBM compatibili dotati di sistema operativo MS-DOS.

ATTENZIONE: Questo documento e' disponibile anche in formato ipertestuale attraverso il nuovo sistema di informazione World Wide Web sviluppato al CERN di Ginevra.

CARATTERISTICHE TECNICHE

Funzionalita' Principali:

  • Registrazione automatica delle transazioni di incasso
  • Gestione separata di contanti e pagamenti con carta di credito
  • Generazione di report giornalieri, settimanali e mensili
  • Controllo automatico di quadratura dei conti
  • Archiviazione sicura dei dati su disco rigido
  • Interfaccia utente a menu guidato
  • Supporto per stampanti a matrice di punti e laser

Specifiche Tecniche:

  • Linguaggio di programmazione: COBOL-85
  • Database: file sequenziali indicizzati
  • Interfaccia: modalita' testo a 80 colonne
  • Capacita' massima: 32.000 transazioni per anno
  • Supporto multi-utente attraverso rete Novell NetWare

VANTAGGI OPERATIVI

Risparmio di Tempo

L'automazione dei processi di registrazione permette di ridurre drasticamente il tempo dedicato alla contabilizzazione degli incassi. Le operazioni che tradizionalmente richiedevano ore di lavoro manuale possono ora essere completate in pochi minuti.

Eliminazione degli Errori

Il controllo automatico di quadratura elimina la possibilita' di errori di calcolo o di trascrizione. Ogni discrepanza viene immediatamente segnalata dal sistema.

Controllo Totale

Ogni transazione viene registrata con data, ora e codice operatore. E' impossibile modificare o cancellare i dati senza lasciare traccia nell'audit trail.

Accesso Multi-Utente

Attraverso una rete locale, il sistema permette l'accesso simultaneo a piu' operatori, ciascuno con i propri privilegi di accesso personalizzati.

REQUISITI DI SISTEMA

Configurazione Minima:

  • Personal Computer IBM compatibile con processore 80286
  • Memoria RAM: 640 KB
  • Disco rigido: 20 MB di spazio disponibile
  • Sistema operativo: MS-DOS versione 3.3 o superiore
  • Monitor: Hercules monocromatico o EGA/VGA
  • Stampante: compatibile Epson o HP LaserJet

Configurazione Consigliata:

  • Personal Computer IBM compatibile con processore 80386
  • Memoria RAM: 2 MB (per installazioni multi-utente)
  • Disco rigido: 40 MB di spazio disponibile
  • Sistema operativo: MS-DOS versione 5.0
  • Monitor: VGA a colori
  • Scheda di rete: Ethernet compatibile Novell
